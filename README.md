# Data-Cleaning-Project-1
A beginner-friendly data analytics project focused on cleaning and analyzing car data with Python

# About This Project

This project was completed as part of my Data Analytics Internship. The goal was to work with a car dataset, identify data quality issues, and prepare the data for further analysis.

## What I Did

I started by exploring the dataset and checking for any missing or incorrect information. During the process, I found some missing values and duplicate records.

To improve the quality of the data, I:

* Removed duplicate entries
* Handled missing values in the Price and Mileage columns
* Checked the dataset for consistency
* Saved the cleaned dataset for future analysis

## Tools Used

* Python
* Pandas
* VS Code
* Excel

## Key Learnings

Through this project, I learned how to:

* Load and work with datasets using Pandas
* Identify and handle missing data
* Remove duplicate records
* Clean and organize data for analysis
* Use Python for real-world data analytics tasks

## Outcome

After cleaning the dataset, the data became more accurate and reliable. The final dataset is ready for further analysis and visualization.
